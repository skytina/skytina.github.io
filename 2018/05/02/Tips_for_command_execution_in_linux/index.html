命令执行绕过技巧-linux环境

当在挖掘命令执行漏洞的时候，总会遇到黑名单，这时候一些绕过技巧就很有用了。

1. 空格过滤了

1.1 Method1:可以使用${IFS}、$IFS

优势:

  • sh、bash都可以

例子:
wooyun-2012-07068

1.2 Method2:可以使用环境变量

优势:

  • sh、bash都可以

1.3 Method3:可以使用{}

缺点:

  • 不能再sh下运行，bash可以
    • sh -c {ls,-al} 运行失败

2. 基于sh或bash

场景1

  • shell=True
    • 可以使用\x0a（换行符）来追加自己的命令

场景2

  • bash环境下，不需要管单引号或者双引号
    • 输入点使用$(id)即可
  • sh环境下
    • 输入点在双引号中，$(id)有效
